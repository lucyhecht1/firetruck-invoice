# Fire Truck Invoice Generator

---

## Demo 1: Happy Customer:
A walkthrough of the complete flow: entering a valid truck listing, generating a PDF, and sending the invoice via email.

---

## Demo 2: Invalid Inputs
See how the service responds to empty or incorrectly formatted fields.

---

## Demo 3: Internal Service Errors
Demonstrates how the service handles unexpected API issues like server errors or missing listings.

---

### Technologies:
- React
- Axios for API calls
- jsPDF to generate PDFs
- EmailJS to send emails
- CSS for styling
